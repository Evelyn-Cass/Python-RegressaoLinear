# 📈 Python-RegressaoLinear

A Python project for performing linear regression analysis, built with Jupyter Notebook. Demonstrates how to model relationships between variables using statistical techniques.

## 📱 About The Project

Python-RegressaoLinear is a simple and practical implementation of linear regression. The project loads a dataset and applies linear regression to find the relationship between variables.

This project was created for study purposes and to demonstrate regression analysis using Python.

- Linear regression model implementation
